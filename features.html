Saphira

Assistente de desktop com IA local para Windows. Converse por texto ou voz, controle aplicativos e veja a tela — tudo roda na sua maquina.

Funcionalidades
💬

Chat Local via LLM

Converse com a Saphira usando texto. O LLM roda localmente via Ollama com o modelo gemma3:4b — zero custo, sem API keys, sem envio de dados para servidores terceiros.

🎙️

Reconhecimento de Voz (STT)

Fale naturalmente e a Saphira entende o que voce disse. Usa RealtimeSTT com modelo Whisper e deteccao automatica de fim de fala (VAD). Modo pontual (/voz) ou contínuo (/voz_continuo).

🔊

Sintese de Voz (TTS)

A Saphira responde falando, usando RealtimeTTS com Azure Speech (voz pt-BR-ThalitaNeural). O TTS e opcional e configuravel via .env.

👁️

Visao de Tela

Peca para a Saphira "olhar" sua tela. O sistema captura a tela com Pillow.ImageGrab e analisa com o modelo Moondream via Ollama, descrevendo o que esta sendo exibido.

💻

Controle de Aplicativos

Abra programas pelo comando de voz ou texto: "abre o chrome", "roda o vscode", etc. O sistema busca executaveis dinamicamente no disco e suporta apelidos configuraveis (navegador, jogos, etc).

🧠

Memoria de Sessao

A conversa e mantida em contexto — a Saphira lembra o que foi dito anteriormente (rolling window configuravel via MAX_TURNS). Comando limpar reseta a memoria da sessao.

Personalidade

A Saphira tem personalidade sarcastica, direta e com humor seco. A persona e definida no Modelfile e embutida no modelo customizado Ollama.

  • Levemente sarcastica, mas sempre util
  • Respostas curtas e diretas — nada de enrolacao
  • Tom descontraido e amigavel
  • Nunca menciona que e uma IA, modelo ou programa
  • Fala em portugues do Brasil por padrao
  • Senso de humor seco e inteligente

Arquitetura
Ai-daughter/ ├── main.py → Ponto de entrada, loop principal interativo ├── Modelfile → Persona "Saphira" embutida no modelo Ollama │ ├── core/ │ ├── orchestrator.py → Detecta intencao e coordena os modulos │ ├── brain.py → Comunicacao com Ollama (LLM) │ └── memory.py → Memoria de conversa (rolling window) │ ├── voice/ │ ├── stt.py → Speech-to-Text via RealtimeSTT + Whisper │ └── tts.py → Text-to-Speech via RealtimeTTS + Azure │ ├── actions/ │ ├── commands.py → Mapeamento de comandos e apelidos de apps │ ├── executor.py → Executa acoes (abre apps) via subprocess │ ├── finder.py → Busca dinamica de .exe no filesystem │ └── vision.py → Captura de tela + analise com Moondream │ └── config/ ├── settings.py → Configuracoes centrais (env, paths, keywords) ├── responses.json → Respostas predefinidas do sistema └── responses.py → Loader de respostas com formatacao
Stack Tecnologico

LLM Local

Ollama com gemma3:4b — processamento 100% local, sem API keys.

Visao

Moondream — modelo de visao leve para analise de imagens via Ollama.

Voz (STT)

RealtimeSTT com modelo Whisper small e deteção automatica de voz (VAD).

Voz (TTS)

RealtimeTTS com Azure Speech (voz Neural brasileira).

Python

3.12+ — type hints, dataclasses, dotenv para configuracao.

Plataforma

Windows 10/11 — captura de tela GDI, busca de executaveis, subprocess.
